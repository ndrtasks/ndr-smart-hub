# NDR Smart Hub - HR Test / Legacy Sources

هذه الحزمة مخصصة للاختبارات فقط.

- جميع الإجراءات والنماذج في هذا المسار قديمة وغير معتمدة للإنتاج.
- لا يجوز لـ NDR AI تقديمها على أنها سياسة أو إجراء حالي معتمد.
- عند استلام الملفات الأصلية المعتمدة: تحذف/تعزل مصادر TEST_LEGACY ثم يعاد بناء الفهرس وربط الخدمات.
- يحتفظ سجل المعاملات التجريبية فقط بمرجع النسخة التي استخدمت أثناء الاختبار.

المجلدات:
- procedures: إجراءات الموارد البشرية القديمة
- forms: النماذج القديمة المتاحة
- guides: دليل نظام الموارد البشرية
- employees: بيانات الموظفين لاختبار الحسابات والهيكل
- document-registry.test.json: سجل وثائق الاختبار
- linkage-audit.test.json: الروابط المكتشفة بين الإجراءات والنماذج
